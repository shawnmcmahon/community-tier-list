# Community Tier Lists MVP Demo Script

## Setup

- Open one host browser window logged in with Twitch.
- Open one viewer browser window or incognito session.
- Keep the host dashboard on screen before entering the session.

## Demo Flow

1. Show the dashboard and create a new session.
2. Paste a TierMaker `create` URL and import the item pool.
3. Point out that the host can also paste manual labels if TierMaker blocks import.
4. Go live and copy the session link.
5. Open the viewer page in the second window.
6. Stage an item from the pool.
7. Open voting and cast a few viewer votes.
8. Show the distribution updating in real time.
9. Finalize the item into a tier and show the streamer and community boards update.
10. Repeat once with a second item.
11. Complete the session.

## Talking Points

- Twitch-authenticated host flow
- direct session links for viewers
- real-time vote distribution and presence
- host-side TierMaker import with graceful failure handling
- separate streamer and community board outcomes
