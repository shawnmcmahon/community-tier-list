# Future Marketing Ideas

## AI Model Tier List Twitch Channel

Potential marketing experiment:

- Create a Twitch channel where multiple AI personas or models vote against each other on preset tier lists.
- Give each model a distinct voice so they can explain ratings, disagree, and roast each other during the stream.
- Let live viewers participate through the same voting flow used by the product.
- Allow the AI personas to react to chat and banter with viewers while the tier list is unfolding.

Why this could work:

- It demonstrates the product in a format that is native to Twitch.
- It creates repeatable content around debates, rankings, and personality-driven interactions.
- It shows both streamer-side and audience-side value in a public setting.

Risks and constraints:

- Running chat-responsive AI agents continuously could become expensive, especially for a 24/7 channel.
- Moderation would need to be deliberate if bots are allowed to roast viewers or react directly to chat.
- The content format should be scoped so it stays entertaining without requiring constant manual intervention.

Possible lower-cost versions:

- Run scheduled events instead of a 24/7 stream.
- Use a smaller set of AI reactions outside the highest-cost portions of the stream.
- Limit direct chat interaction to specific segments or premium showcase sessions.
